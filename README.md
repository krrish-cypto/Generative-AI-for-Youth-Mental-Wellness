Aura: Your Mental Wellness Companion (Prototype)
This repository contains a web-based, functional prototype for Aura, a compassionate Gen AI-powered chatbot designed to be a personal mental wellness companion for young people. This prototype was created as part of the GenAI Exchange Hackathon.

🌟 Project Vision
Aura's mission is to provide an accessible, confidential, and safe space for youth to explore their feelings, learn coping strategies, and practice mindfulness. By leveraging advanced Generative AI, Aura offers personalized, empathetic conversations that focus on proactive wellness.

🚀 About This Prototype
This prototype is a single-file web application built with HTML, Tailwind CSS, and JavaScript. It is designed to demonstrate the core user experience and the AI-powered chat functionality. The goal is to showcase how an empathetic conversation with Aura would feel.

Core Features Implemented
Empathetic Chat: Engage in a real-time conversation with Aura. The backend is powered by a generative AI model (via an API call) that is instructed to be a compassionate and supportive companion.

Intuitive UI: A clean, mobile-first chat interface that is welcoming and easy to use.

Clear Conversation: A button to easily clear the chat history and start a new conversation.

Simulated Features
To illustrate the full vision, the following features are simulated with pre-defined responses when their corresponding buttons are clicked:

🌬️ Breathing Exercise: Simulates the start of a guided mindfulness session.

📓 Mood Journal: Simulates a prompt for the user to start journaling.

📚 Resource Hub: Simulates Aura offering to share helpful articles and resources.

💻 Technology Stack
Frontend: HTML5, Tailwind CSS

Logic: Vanilla JavaScript

Generative AI: The prototype makes live calls to a generative AI API to power Aura's responses, guided by a carefully crafted system prompt that defines its persona.

▶️ How to Run the Prototype
No complex setup is required!

Download the File: Save Generative-AI-for-Youth-Mental-Wellness
.html file to your local machine.

Open in Browser: Open the file directly in any modern web browser (like Google Chrome, Mozilla Firefox, or Microsoft Edge).

Start Chatting: The chat interface will load, and you can begin your conversation with Aura immediately.

🔮 Future Scope
This prototype is the first step. Based on our project proposal, future development could include:

Full Feature Integration: Building out the guided exercises, mood journaling, and resource hub with dynamic content.

Database Integration: Using a database like Firestore to save conversation history and mood data for a truly personalized experience.

Multi-language Support: Expanding Aura's reach to a global audience.

Voice Interaction: Allowing users to speak with Aura directly.

Gamification: Adding engagement elements like badges and streaks.
